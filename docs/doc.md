### 协议

**\* 协议字段均采用小端序**

#### 协议头

```
 0               1               2               3
 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7
+---------------------------------------------------------------+
| MessageType   |                 Messages...                   |
+---------------------------------------------------------------+

```

#### MessageType

通过uint8表示消息的类型

##### 1 ：世界位置原始信息

消息体如下：

```
摄像机     

 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7
+---------------+-------------------------------+
|      self     |   Number of players(uint16)   | 
+---------------+-------------------------------+
*self 表示是否存在玩家自己，存在(1)则为玩家列表第一个,不存在(0)
玩家1 玩家2 ...
```

**摄像机:**

- float: 摄像机中心坐标 X
- float: 摄像机中心坐标 Y
- uint16:一米表示的像素宽度

**玩家：**

- int32: 玩家的血量

- int32: 玩家的血量上限

- int32: 玩家的金币
- int32: 玩家的得分

- [几何体](#shape): 代表玩家的几何体



**几何体<a name="shape"></a>：**

- 类型:
- 通过uint8表示物体的类型

-   - 0 : 圆形
    - 1 : 正三角形
    - 2 : 正方形
    - 3 : 长方形
    - 4 : 正五边形

-   float: 物体质心的 X 轴坐标值

-   float: 物体质心的 Y 轴坐标值

-   float: 物体质心的旋转角度(弧度制)

-   float: 物体的属性	


| 形状                 | 属性                      |
| -------------------- | ------------------------- |
| 圆形                 | 半径(float)               |
| 正三角形             | 边长(float)               |
| 正方形               | 边长(float)               |
| 长方形               | 宽度(float) ，高度(float) |
| 正五边形(用圆形近似) | 内切圆半径(float)         |


##### 2 ：玩家交互信息

消息体如下：

```
 0               1               2               3
 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7 0 1 2 3 4 5 6 7
+---------------------------------------------------------------+
| OperationType |                 messages...                   |
+---------------------------------------------------------------+
```
**OperationType: **

通过uint8表示交互的类型

- 0 : 鼠标移动：(X，Y)(int16，int16)
- 1 : 鼠标按下(发射子弹)：(X，Y)(int16，int16)
- 2 : 键盘按下：String
- 3 : 键盘抬起：String
- 4 : 新建初始角色：无参数
